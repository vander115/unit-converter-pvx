## ✅ Sobre o projeto

Esse projeto foi elaborado como trabalho final da disciplina de Programação Computacional na Universidade Federal do Ceará

O sistema foi contruído em C utilizando apenas funções nativas da linguagem ou que foram criadas por nós

Ele converte as principais unidades de medidas do cotidiano e armazena os resultados em um Histórico
